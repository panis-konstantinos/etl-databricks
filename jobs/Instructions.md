# Overview
(AdventureWorks_Ingestion_Job.yaml)[jobs/AdventureWorks_Ingestion_Job.yaml] constitutes a template to create a job in Databricks to automate the creation of all medallion layers (bronze, silver, gold).

# Steps
1. Create a job in Databricks

2. Select 'Edit as YAML' in the options pane

3. Replace the existing YAML with a copy of (AdventureWorks_Ingestion_Job.yaml)[./AdventureWorks_Ingestion_Job.yaml]

4. In lines 12, 14, 16, 20, 26, and 32, of the YAML, replace placeholders values  with your workspace specific values

5. Click save

6. Return to project's (README)[../README.md]
